# **App Name**: NearRemind

## Core Features:

- Spatial Dashboard: A streamlined list view that prioritizes reminders based on their current proximity to your geographic location.
- Smart Task Categorizer: An AI tool that uses natural language processing to suggest relevant location tags for reminders based on their content.
- Radius Selector: Interface to precisely set triggering radiuses for location alerts on an interactive map component.
- Adaptive Geo-Form: A minimal screen for adding reminders that uses address autocompletion to quickly bind tasks to specific locations.
- Proximity Engine: Background service utilizing the Geolocation API to check device distance from active task pins.
- Contextual Preferences: Settings portal to manage notification frequency and global location accuracy levels.

## Style Guidelines:

- Primary Color: Electric Indigo (#5d52fa), chosen to evoke a sense of digital precision and high technology.
- Background: Deep Night (#12111c), a dark scheme that emphasizes focus and reduces eye strain.
- Accent Color: Polar Azure (#5caaff), providing high contrast against the dark background for active UI states.
- Font Pairing: 'Space Grotesk' for modern, tech-forward headlines and 'Inter' for a clean, legible sans-serif body experience.
- Minimalist card-based structure with generous whitespace and 16px border radii to create a contemporary mobile feel.
- Ultra-thin line icons with subtle Indigo glows for wayfinding and navigation.
- Soft micro-interactions such as vertical spring-animations when sorting reminders by proximity.
